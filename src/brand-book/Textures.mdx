### Blur Levels

| Nível | Valor | Uso |
|-------|-------|-----|
| `blur-xs` | 2px | Sutil, quase imperceptível |
| `blur-sm` | 4px | Leve separação |
| `blur` | 8px | Padrão para cards |
| `blur-md` | 12px | Destaque moderado |
| `blur-lg` | 16px | Forte separação |
| `blur-xl` | 24px | Overlay dramático |

## Do's and Don'ts

### Do's
- Use gradientes para criar hierarquia visual
- Aplique glass morphism sobre fundos coloridos ou imagens
- Use elementos decorativos com baixa opacidade (10-20%)
- Mantenha o texto legível sobre texturas

### Don'ts
- Não use glass morphism sobre fundos brancos simples
- Não sobreponha muitos elementos decorativos
- Evite gradientes muito contrastantes para textos longos
- Não use blur excessivo que comprometa a performance
