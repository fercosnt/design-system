### Area de Protecao

O espaco ao redor do logo deve ser no minimo equivalente a altura do simbolo (isotipo) em todos os lados.

## Download de Assets

Para acessar todos os arquivos de logo, consulte a pasta `src/assets/logos/` do projeto.

```tsx
// Importar logos via TypeScript
import { logoSpecs, logoAssets } from '@/assets/logos'

// Exemplo de uso
<img src={logoAssets.horizontal.azul} alt="Beauty Smile" />
```

### Estrutura de Arquivos

```
src/assets/logos/
├── svg/
│   ├── horizontal/      # 8 arquivos
│   ├── vertical/        # 13 arquivos
│   ├── isotipo/         # 12 arquivos
│   └── tipografico/     # 8 arquivos
├── 3d-png/              # 15 arquivos
├── 3d-webp/             # 17 arquivos
└── index.ts             # Exports tipados
```
