# Templates Gallery

Galeria de templates pré-construídos para acelerar o desenvolvimento de aplicações Beauty Smile.

---

## Visão Geral

Os templates são layouts completos e funcionais que combinam múltiplos componentes do Design System.
Eles servem como ponto de partida para páginas comuns e podem ser customizados conforme necessário.

### Categorias

| Categoria | Descrição | Tema |
|-----------|-----------|------|
| **Admin** | Área administrativa, backoffice | Azul profundo (Primary) |
| **Public** | Área pública, cliente | Turquesa (Accent) |

## Como Usar

### Importando um Template

```tsx
import { DashboardAdminTemplate } from '@/templates/admin'

function AdminDashboard() {
  return (
    <DashboardAdminTemplate
      title="Dashboard"
      userName="Dr. João Silva"
      metrics={[
        { title: 'Pacientes', value: '1,234' },
        { title: 'Consultas Hoje', value: '12' },
      ]}
      navItems={[
        { label: 'Dashboard', isActive: true },
        { label: 'Pacientes' },
        { label: 'Configurações' },
      ]}
    >
      {/* Seu conteúdo */}
    </DashboardAdminTemplate>
  )
}
```

### Customizando Templates

Os templates aceitam diversas props para customização:

1. **Conteúdo**: Passe elementos via `children` ou props específicas
2. **Navegação**: Configure itens de menu via `navItems`
3. **Ações**: Adicione callbacks como `onSubmit`, `onCancel`, etc.
4. **Estilo**: Use `className` para estilos adicionais

## Princípios de Design

### Consistência
Todos os templates seguem os mesmos padrões de:
- Espaçamento (scale de 4px)
- Tipografia (Inter para texto, Montserrat para headings)
- Cores (paleta brand)
- Componentes (do Design System)

### Responsividade
Templates são responsivos por padrão:
- Mobile: Layout em coluna única
- Tablet: Layouts adaptados
- Desktop: Layouts completos com sidebar

### Acessibilidade
- Navegação por teclado
- Estrutura semântica de headings
- Labels em formulários
- Contraste adequado
