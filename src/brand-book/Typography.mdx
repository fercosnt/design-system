## Letter Spacing

| Token | Valor | Uso |
|-------|-------|-----|
| `tracking-tighter` | -0.05em | Headings grandes |
| `tracking-tight` | -0.025em | Headings |
| `tracking-normal` | 0 | Corpo de texto |
| `tracking-wide` | 0.025em | Labels, botoes |
| `tracking-wider` | 0.05em | Caps, destaques |
| `tracking-widest` | 0.1em | All caps |

---

## Line Height

| Token | Valor | Uso |
|-------|-------|-----|
| `leading-none` | 1 | Headings grandes |
| `leading-tight` | 1.25 | Headings |
| `leading-snug` | 1.375 | Subtitulos |
| `leading-normal` | 1.5 | Corpo de texto |
| `leading-relaxed` | 1.625 | Texto longo |
| `leading-loose` | 2 | Espacamento generoso |

## Uso com Tailwind

```tsx
// Headings
<h1 className="font-heading text-5xl font-bold tracking-tight">
  Titulo Principal
</h1>

// Body text
<p className="font-sans text-base leading-normal">
  Texto do paragrafo com configuracao padrao.
</p>

// Labels
<label className="font-sans text-sm font-medium tracking-wide">
  Label do Campo
</label>

// Code
<code className="font-mono text-sm">
  console.log('Hello')
</code>
```

## Tokens TypeScript

```typescript
import { typography } from '@/tokens/typography'

// Familias
typography.fontFamily.sans    // Inter stack
typography.fontFamily.heading // Montserrat stack
typography.fontFamily.mono    // JetBrains Mono stack

// Tamanhos
typography.fontSize.base      // ['1rem', { lineHeight: '1.5rem' }]
typography.fontSize['2xl']    // ['1.5rem', { lineHeight: '2rem' }]

// Pesos
typography.fontWeight.semibold // '600'
typography.fontWeight.bold     // '700'

// Headings pre-configurados
typography.headings.h1 // { fontFamily, fontSize, lineHeight, fontWeight, letterSpacing }
```
