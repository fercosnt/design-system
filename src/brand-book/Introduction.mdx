# Beauty Smile Design System

Bem-vindo ao Design System oficial da Beauty Smile, criado para garantir consistencia visual e experiencial em todas as plataformas e pontos de contato da marca.

## Sobre a Beauty Smile

A Beauty Smile e uma clinica odontologica premium especializada em harmonizacao orofacial e estetica dental. Nossa missao e transformar sorrisos e elevar a autoestima de nossos pacientes atraves de tratamentos de excelencia.

## Proposito do Design System

Este Design System serve como fonte unica de verdade para:

- **Designers**: Especificacoes visuais, tokens e guidelines
- **Desenvolvedores**: Componentes prontos para uso, codigo e documentacao
- **Marketing**: Diretrizes de marca, logos e tom de voz
- **Stakeholders**: Referencia visual da identidade da marca

## Arquetipos de Marca

A Beauty Smile combina dois arquetipos complementares:

### O Mago (Primario)
- **Essencia**: Transformacao e resultados surpreendentes
- **Promessa**: "Vou transformar seu sorriso e sua autoestima"
- **Tom**: Confiante, inspirador, visionario

### O Prestativo (Secundario)
- **Essencia**: Cuidado e acolhimento
- **Promessa**: "Voce esta em boas maos"
- **Tom**: Acolhedor, atencioso, confiavel

## Como Usar Este Design System

### Para Designers

1. Consulte os tokens de design para especificacoes
2. Use os componentes documentados como referencia
3. Siga as guidelines de espacamento e tipografia
4. Mantenha a consistencia visual em todos os layouts

### Para Desenvolvedores

```bash
# Instalar dependencias
npm install @beautysmile/design-system

# Importar componentes
import { Button, Card, GlassPanel } from '@beautysmile/design-system'

# Importar tokens
import { colors, typography, spacing } from '@beautysmile/design-system/tokens'
```

### Para Marketing

1. Baixe logos em diferentes formatos e cores
2. Consulte o Brand Voice para comunicacao
3. Use as cores oficiais da marca
4. Siga as guidelines de aplicacao do logo
