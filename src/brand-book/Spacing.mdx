## Z-Index Scale

| Token | Valor | Uso |
|-------|-------|-----|
| `z-base` | 0 | Elementos base |
| `z-dropdown` | 1000 | Dropdowns, selects |
| `z-sticky` | 1100 | Headers fixos |
| `z-fixed` | 1200 | Elementos fixed |
| `z-modal-backdrop` | 1300 | Backdrop de modais |
| `z-modal` | 1400 | Modais |
| `z-popover` | 1500 | Popovers |
| `z-tooltip` | 1600 | Tooltips |
| `z-notification` | 1700 | Notificacoes |
| `z-top` | 9999 | Acima de tudo |

## Uso com Tailwind

```tsx
// Padding e margin
<div className="p-4 m-2">16px padding, 8px margin</div>
<div className="px-6 py-4">24px horizontal, 16px vertical</div>

// Gap
<div className="flex gap-4">16px entre items</div>
<div className="grid gap-6">24px grid gap</div>

// Border radius
<div className="rounded-lg">16px radius</div>
<div className="rounded-full">Pill shape</div>

// Responsive
<div className="p-4 md:p-6 lg:p-8">
  16px mobile, 24px tablet, 32px desktop
</div>
```

## Tokens TypeScript

```typescript
import { spacingTokens } from '@/tokens/spacing'

// Escala base
spacingTokens.spacing[4]   // '16px'
spacingTokens.spacing[8]   // '32px'

// Semantico
spacingTokens.semanticSpacing.md         // '16px'
spacingTokens.semanticSpacing.cardPadding // '24px'

// Border radius
spacingTokens.borderRadius.lg   // '16px'
spacingTokens.borderRadius.full // '9999px'

// Breakpoints
spacingTokens.breakpoints.md  // '768px'
spacingTokens.breakpoints.xl  // '1280px'

// Z-index
spacingTokens.zIndex.modal    // 1400
spacingTokens.zIndex.tooltip  // 1600
```
