## Especificacoes das Sombras

| Token | Valor CSS | Uso |
|-------|-----------|-----|
| `shadow-sm` | `0 1px 2px 0 rgba(45,46,48,0.05)` | Inputs, cards sutis |
| `shadow` | `0 1px 3px 0 rgba(45,46,48,0.1)...` | Cards padrao |
| `shadow-md` | `0 4px 6px -1px rgba(45,46,48,0.1)...` | Popovers, tooltips |
| `shadow-lg` | `0 10px 15px -3px rgba(45,46,48,0.1)...` | Modais, dialogs |
| `shadow-xl` | `0 20px 25px -5px rgba(45,46,48,0.1)...` | Overlays importantes |
| `shadow-2xl` | `0 25px 50px -12px rgba(45,46,48,0.25)` | Hero, CTAs |
| `shadow-inner` | `inset 0 2px 4px 0 rgba(45,46,48,0.05)` | Inputs focus |

## Uso com Tailwind

```tsx
// Sombras padrao
<div className="shadow-sm">Sombra leve</div>
<div className="shadow-md">Sombra media</div>
<div className="shadow-lg">Sombra grande</div>

// Hover effects
<div className="shadow-md hover:shadow-lg transition-shadow">
  Card com hover
</div>

// Focus ring
<button className="focus:ring-2 focus:ring-primary/30">
  Button com focus ring
</button>

// Glass effect
<div className="bg-white/15 backdrop-blur-lg shadow-glass">
  Glass Card
</div>
```

## Tokens TypeScript

```typescript
import { shadows } from '@/tokens/shadows'

// Box shadows
shadows.boxShadow.sm      // '0 1px 2px 0 rgba(45, 46, 48, 0.05)'
shadows.boxShadow.lg      // '0 10px 15px -3px rgba(45, 46, 48, 0.1)...'
shadows.boxShadow.inner   // 'inset 0 2px 4px 0 rgba(45, 46, 48, 0.05)'

// Glass shadows
shadows.glassShadow.DEFAULT // '0 4px 6px 0 rgba(0, 0, 0, 0.1)...'

// Colored shadows
shadows.coloredShadow.primary.DEFAULT // '0 4px 6px 0 rgba(0, 16, 158, 0.3)'
shadows.coloredShadow.accent.lg       // '0 10px 15px -3px rgba(53, 191, 173, 0.4)'

// Focus shadows
shadows.semanticShadow.focus.primary // '0 0 0 3px rgba(0, 16, 158, 0.3)'

// Glow effects
shadows.semanticShadow.glow.accent   // '0 0 20px rgba(53, 191, 173, 0.5)'

// Elevation system
shadows.elevation.level0 // boxShadow.none
shadows.elevation.level2 // boxShadow.md
shadows.elevation.level4 // boxShadow.xl
```
