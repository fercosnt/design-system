# Beauty Smile Design System

Welcome to the **Beauty Smile Design System** ‚Äî a comprehensive React component library featuring modern UI components, design tokens, and two distinct themes for building beautiful, accessible, and consistent user interfaces.

## ‚ú® What's Inside

### üé® Design Tokens

Centralized design decisions that ensure visual consistency across all projects:

- **Colors** - Brand colors (Deep Blue, Turquoise, Gold), neutral scale, semantic colors
- **Typography** - Font families (Inter, Montserrat), sizes, weights, line heights
- **Spacing** - Consistent 4px-based scale (0-32)
- **Shadows** - 5-level elevation system + glass morphism effects
- **Animations** - Keyframes, durations, and timing functions

[View Design Tokens ‚Üí](/?path=/docs/design-tokens-overview--docs)

### üß© Components

60+ production-ready React components built with:

- **Radix UI** - Accessible component primitives
- **TypeScript** - Full type safety
- **Tailwind CSS** - Utility-first styling
- **Responsive** - Mobile-first design

**Component Categories:**
- **Atoms** - Basic building blocks (Button, Input, Badge, etc.)
- **Molecules** - Composite components (Card, Modal, Dropdown, etc.)
- **Organisms** - Complex components (Navigation, Header, Sidebar, etc.)

[Browse Components ‚Üí](/?path=/docs/components-atoms-button--docs)

### üìê Templates

Pre-built page templates for common use cases:

- **Admin Dashboard** - Deep Blue theme for internal tools

## üöÄ Quick Start

### Installation

```bash
npm install @beautysmile/design-system
```

### Basic Usage

```tsx
import { Button, Card } from '@beautysmile/design-system/components'
import { colors } from '@beautysmile/design-system/tokens'
import '@beautysmile/design-system/styles'

function App() {
  return (
    <Card className="p-6">
      <h1 style={{ color: colors.primary.DEFAULT }}>
        Hello Beauty Smile!
      </h1>
      <Button variant="primary">
        Get Started
      </Button>
    </Card>
  )
}
```

## üé® Design Philosophy

### Accessibility First
All components are built with accessibility in mind:
- WCAG 2.1 AA compliance
- Keyboard navigation support
- Screen reader friendly
- ARIA attributes
- Focus management

### Modern & Clean
- Minimal, uncluttered interfaces
- Consistent spacing and alignment
- Thoughtful color usage
- Smooth animations and transitions

### Developer Experience
- TypeScript for type safety
- Tree-shakeable imports
- Comprehensive documentation
- Copy-paste code examples
- IntelliSense support
